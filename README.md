# QwenPaw Agent Skills

QwenPaw 智能体技能合集，开箱即用。

## 技能列表

| 技能 | 说明 |
|:--|:--|
| [hot-news](skills/hot-news/) | 热点新闻采集推送（人民网、新华网、GitHub周榜、微博热搜） |

## 安装方式

```bash
# 克隆仓库
git clone https://github.com/gjz518/qwenpaw-skills.git

# 将技能目录复制到工作区 skills/ 目录
cp -r qwenpaw-skills/skills/* /path/to/workspace/skills/
```
